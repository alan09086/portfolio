Featured Project

Supply Chain Multi-Agent System

LangGraph + LlamaIndex RAG + FAISS + FastAPI + Streamlit + Langfuse

Codeberg GitHub

A production-grade multi-agent system for CPG supply chain optimization. Three specialized AI agents — demand forecasting, inventory optimization, and supplier risk — collaborate through a LangGraph state machine with typed state, conditional human-in-the-loop approval via interrupt(), and full observability through self-hosted Langfuse.
