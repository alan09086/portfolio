Industrial AI: Rung Reader

Rung Reader

AI-powered Allen-Bradley PLC ladder logic troubleshooter

Codeberg

When a PLC faults on a production line, an operator reads ladder logic in Studio 5000, traces back from a failed output through every condition until they find the false rung, then walks the floor checking sensors and wires. Rung Reader automates the trace — given an L5X export and live tag values, it produces the root-cause analysis a senior maintenance tech would, narrates it in plain English, surfaces relevant SOPs from a plant document corpus, and proactively flags anomalies.

Design principle: The core trace is deterministic — a pure-function rung walker on the L5X AST. The LLM only narrates what the walker found; it never reasons about ladder semantics from scratch. During the M0 spike, llama3.1:8b claimed XIC means "not in contact" (it actually means "examine if closed"). LLMs hallucinate domain knowledge; the walker doesn't.

Systems: IronTide

IronTide

From-scratch Rust BitTorrent engine — libtorrent-rasterbar feature parity

Codeberg GitHub

A full BitTorrent engine written in Rust from first principles, targeting parity with libtorrent-rasterbar — the 15-year C++ reference library behind qBittorrent, Deluge, and Transmission. IronTide ships a 17-crate modular workspace, 31 BEP protocol implementations, async actor architecture (tokio select! loops with backpressure), pluggable backends for crypto and disk I/O, a Slint desktop GUI, and an axum HTTP REST API with an HTMX web UI.
